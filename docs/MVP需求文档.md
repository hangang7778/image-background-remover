## 1. 产品概述

### 1.1 产品名称
**BG Remover**

### 1.2 核心功能
一键移除图片背景，下载透明PNG

### 1.3 目标用户
| 用户群体 | 使用场景 |
|---------|---------|
| 设计师 | 产品图、宣传物料去背景 |
| 电商卖家 | 淘宝、Shopify 商品图处理 |
| 自媒体创作者 | 头像、封面图制作 |
| 普通用户 | 证件照换底色等日常需求 |

### 1.4 核心价值主张
- ⚡ **快速** - 5秒内完成背景移除
- 💚 **免费** - 每日免费额度
- 🔒 **安全** - 图片不存储，用完即弃
- 📱 **简单** - 无需注册，上传即用

## 2. 技术方案

### 2.1 系统架构

```
┌─────────────┐     ┌──────────────┐     ┌───────────────┐     ┌──────────────┐
│  用户浏览器  │────▶│ Cloudflare  │────▶│ Cloudflare   │────▶│ Remove.bg   │
│  (上传图片)  │     │   Pages     │     │   Workers    │     │    API      │
└─────────────┘     │   (前端)     │     │   (后端)     │     │  (图像处理)   │
                    └──────────────┘     └───────────────┘     └──────────────┘
```

### 2.2 技术栈

| 层级 | 技术选型 | 说明 |
|-----|---------|------|
| 前端框架 | Next.js 14 | App Router, SSR/SSG |
| UI 样式 | Tailwind CSS | 响应式设计 |
| 后端运行时 | Cloudflare Workers | 边缘计算，按请求计费 |
| 图像处理 | Remove.bg API | AI背景移除 |
| 部署平台 | Cloudflare Pages | 免费静态托管 |
| 域名 | 自定义域名（可选） | 用于正式上线 |

### 2.3 数据流程

1. 用户拖拽/选择图片（前端校验格式和大小）
2. 图片以 FormData 形式 POST 到 `/api/remove-bg`
3. API Route 将图片转发给 Remove.bg API
4. Remove.bg 返回透明PNG
5. API Route 将结果转递给前端
6. 前端展示对比图，提供下载按钮

### 2.4 关键技术点

- **图片不落地**：使用内存流处理，不存储
- **边缘计算**：Cloudflare Workers 全球低延迟
- **按需计费**：Remove.bg $0.09/张，免费额度50张/月

## 3. 功能清单

### 3.1 MVP 必须功能（P0）

| 功能 | 描述 | 验收标准 |
|-----|------|---------|
| F01 | 图片上传 | 支持拖拽和点击上传 |
| F02 | 格式校验 | 仅接受 JPG/PNG/WebP |
| F03 | 大小限制 | 单文件 ≤ 10MB |

| F06 | 一键下载 | 下载透明PNG格式 |
| F07 | 移动端适配 | 手机/平板可正常使用 |
| F08 | 每日限制 | 每人每天 5 次免费额度 |

| 批量处理 | P2 | 一次处理多张图片 |
| 历史记录 | P2 | 查看历史处理记录 |
| 格式转换 | P2 | 透明背景→白底/色底 |

| 付费订阅 | P3 | 解锁更多额度 |

## 5. 非功能需求

### 5.1 性能要求

| 指标 | 目标值 |
|-----|-------|
| 首屏加载 | < 2秒 |
| 图像处理 | < 5秒（取决于图片大小） |
| 并发支持 | 100+ 同时请求 |

### 5.2 安全要求

- 图片不存储在服务器
- API 频率限制防滥用
- 移除.bg API Key 仅存于后端环境变量
- 前端不暴露任何敏感信息

### 5.3 隐私要求

- 不收集用户个人信息
- 不存储上传/处理后的图片
- 处理完成立即丢弃内存数据
